### 지원 타입

- 인터페이스 : annotation
- 클래스 : 열거 타입
  - 멤버 : 필드, 메서드, 멤버 클래스, 멤버 인터페이스
  - 메서드 시그니처 : 메서드 이름, 매개변수 (반환값의 타입은 포함 x)
  - 상속 : 클래스가 인터페이스 구현 or 인터페이스가 다른 인터페이스 확장
  - 패키지 접근
- 배열
- 기본 타입

API : 프로그래머가 클래스, 인퍼테이스, 패키지를 통해 접근할 수 있는 형태
